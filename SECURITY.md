# Security Policy

## Reporting a vulnerability in this repo

If you find a security issue in this repository itself (not in code you audited using this tool), please report it responsibly.

**Do not open a public GitHub issue for security vulnerabilities.**

Report it through our confidential disclosure form: https://ironsights.com.au/security-disclosure

Include:
- A description of the vulnerability
- Steps to reproduce
- Potential impact

We will acknowledge receipt within 2 Australian business days and aim to resolve confirmed issues within 14 days.

## Found a vulnerability in your own codebase?

If you used this tool and found a critical issue in your application, [contact IronSights](https://ironsights.com.au/contact) for help with remediation. We offer emergency response for active incidents.
